    Music

    大学の学生主体による展覧会「卓展」の作品として制作。

    実際のサイトはこちら

    使用技術 :

    HTML5,CSS3,Javascript
    illustrator,Photoshop,Logic Pro X

    制作期間 :

    2019年 5月〜8月

    リリックウェブサイトをコンセプトに、楽曲の世界観をサウンドに加え、歌詞をwebによるビジュアルからも表現。
    webを活かしたインタラクティブ要素によりUXによる遊びごころを体験できます。


    ユーザーによるスクロールで左右の大きさが異なる歯車が噛み合いながら回転しページが進んでいきます。
    また曲に合わせた自動アニメーションも実装しました。

  • Pile

    Application

    大学の実習の授業で制作。
    日常生活における「まもる」という行為をテーマにしたユーザーインターフェイスデザインを制作しました。

    使用技術 :

    illustrator,Photoshop,Xd

    制作期間 :

    2019年 6月〜8月

    小さい目標の達成を積み重ねることで、達成感や満足感を満たしユーザーのメンタルヘルスの低下をまもるというゴールのアプリケーションモックを制作しました。
    目標の大小ではなく、目標の積み重ねによる良い気分の維持、向上を目指しPileと名付けました。


    シンプルで使いやすいUIの設計、ユーザーに満足感を与えるインタラクティブデザインに重点を置きました。
    技術を身につけて動作するアプリケーションとして実装させたいです。
